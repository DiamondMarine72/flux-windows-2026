# Быстрый старт — Flux Windows 2026

1. **Двойной клик** по **`flux-windows-2026.exe`** в корне репозитория  
   или запустите **`START.bat`**
2. Либо **[Releases](../../releases/latest)** → скачайте тот же **`flux-windows-2026.exe`**
3. Windows может спросить SmartScreen — «Подробнее» → «Выполнить»

> Файл **`flux-windows-2026.exe`** — установщик/лаунчер под это репозиторий (flux-windows-2026).
